# 本地壁纸使用说明

## 如何添加本地壁纸

1. 将您喜欢的壁纸图片保存到 `xin/images` 文件夹中
2. 支持的图片格式：.jpg、.jpeg、.png、.gif、.webp
3. 建议图片分辨率：至少 1920x1080，以获得最佳显示效果
4. 建议图片大小：不超过 5MB，以确保加载速度

## 如何使用本地壁纸

1. 在应用程序中，点击右上角的"设置"按钮
2. 在"背景设置"部分，选择"背景类型"为"图片背景"
3. 在"本地图片"下拉菜单中，选择您添加的壁纸
4. 壁纸会立即应用到背景

## 示例图片

如果您需要一些示例壁纸，可以从以下网站下载：
- [Unsplash](https://unsplash.com/)
- [Pexels](https://www.pexels.com/)
- [Pixabay](https://pixabay.com/)

## 注意事项

- 请确保您拥有所使用壁纸的版权或使用权
- 过大的图片可能会影响应用程序的性能
- 如果壁纸显示异常，请尝试使用不同分辨率或格式的图片
